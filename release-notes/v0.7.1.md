## What's New

- Added managed daemon cleanup so NeoCode prunes older installed `neocoded` versions after a successful matching install.
- Added a full `just release-dry-run X.Y.Z` workflow to validate notarization, appcast generation, and daemon packaging without publishing a release.
- Improved daemon install/update handling and release validation so the app can recover more cleanly when the daemon is missing or out of date.

## Improvements

- Simplified the flaky transcript UI fixture check so it no longer blocks release validation.
- Updated build and release docs to cover the daemon-aware workflow, artifact naming, and dry-run path.
- Tightened release asset handling so only version-matched daemon archives are uploaded.

## What's New

- Added localization support with initial translations for English, Spanish, Portuguese, French, and Italian, so NeoCode feels much more at home for more users.
- Added in-app todo list support to make longer agent tasks easier to track and manage from the desktop client.
- Improved session reliability with fixes for stuck working states, tool-deadlock failures, and sidebar ordering so active conversations stay easier to follow.

## Improvements

- Fixed text-selection issues in the composer and other text fields.
- Tightened transcript layout polish, including excess bottom padding and thread presentation details.
- Continued shipping polish across session handling and conversation updates to make day-to-day use feel steadier.

## What's New

- NeoCode returns to the simpler in-app runtime flow, removing the external daemon architecture introduced in 0.7.x that left the app in a broken state.
- Sessions and thread startup are more predictable again, while keeping the improved queued-message composer UI and the restored new-thread empty state.

## Improvements

- Reduced release risk by cutting the backend split out of the shipping app and returning to the more stable pre-0.7 architecture.
- Preserved the best recent conversation polish without carrying forward the daemon-specific settings, diagnostics, and runtime complexity.
